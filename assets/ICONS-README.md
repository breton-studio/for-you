# Extension Icons

For the demo, you need to create icon PNG files in three sizes:
- icon-16.png (16x16 pixels)
- icon-48.png (48x48 pixels)
- icon-128.png (128x128 pixels)

## Quick Creation Method

You can use any of these methods:

### Method 1: Using an online tool
1. Go to https://www.favicon-generator.org/ or similar
2. Upload the icon.svg file
3. Generate and download PNG files at 16px, 48px, and 128px

### Method 2: Using Figma/Photoshop
1. Create a 128x128 canvas with the gradient background (#4ade80 to #22c55e)
2. Add the ✨ emoji or sparkle icon in white
3. Export at 128x128, 48x48, and 16x16

### Method 3: Simple solid color (fastest for testing)
Create simple green squares:
- Background: #4ade80 (green)
- Add white ✨ text/emoji if possible

The icons are not critical for functionality testing - Chrome will show a default icon if these are missing.
