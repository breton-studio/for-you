## If Something's Wrong

### Module doesn't appear?
- Wait 3 seconds (might be slow)
- Hard refresh: Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows)
- Check console (F12) for errors

### Quiz doesn't open?
- Make sure toggle is OFF first
- Click the extension icon in toolbar instead
- Reload extension at chrome://extensions/

### No personalization visible?
- Open DevTools console (F12)
- Look for "Applying personalization" message
- Check if site structure matches (DOM selectors may need updating)

### Extension won't load?
- Check all files are present
- Look for red errors at chrome://extensions/
- Try removing and re-adding extension
